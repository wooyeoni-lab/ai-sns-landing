# AI SNS 마케팅 랜딩 페이지

포미서비스 FORMEAI의 AI 기반 SNS 마케팅 랜딩 페이지입니다.

## 프로젝트 소개

AI 기술을 활용한 SNS 마케팅 솔루션을 소개하는 랜딩 페이지입니다.

## 주요 기능

- 반응형 디자인
- 접근성 고려 (ARIA 라벨, 시맨틱 HTML)
- 개인정보처리방침 페이지 포함

## 파일 구조

- `index.html` - 메인 랜딩 페이지
- `privacy.html` - 개인정보처리방침 페이지
